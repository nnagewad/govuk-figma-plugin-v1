# GOV.UK Figma Plugin - V1

## About
This plugin takes a GOV.UK page URL and automatically recreates it as a Figma frame using real design system components.

## How it works
1. Screenshot — it fetches a full-page screenshot of the URL using thum.io, a third-party screenshot service. This captures everything on the page from header to footer.
2. AI vision — the screenshot is sent to Claude (via your Cloudflare Worker proxy) along with a reference list of every available GOV.UK component and its CSS class name. Claude looks at the screenshot and returns a JSON list of which components it can see, in order from top to bottom. 
3. Component matching — the plugin looks up each identified class in a pre-built map (component-map.json) that links CSS classes to actual Figma component keys. This map was generated by scanner.js, which scanned both the GOV.UK Design System and GOV.UK Publishing Components Figma libraries.
4. Frame building — the matched components are imported from the Figma libraries and placed into a 1440px AutoLayout frame, structured to reflect the page layout: header, navigation, and phase banner at the top; main content and sidebar in a two-column body; footer at the bottom.

### Limitation
The limitation is that Claude is making an educated guess from a screenshot, so it can miss components that are subtle, off-screen, or visually ambiguous — which is why the cherry plugin exists.

## How to run
1. Download this repo
2. Open Figma (ideally desktop version)
3. Select *Plugins* in the menu
4. Hover over *Development*
5. Choose the option *Import plugin from manifest*
6. Select the folder of downloaded repo
7. Select `manifest.json`
8. A pop-up should appear that contains all the plugins you've installed
9. Select *GOV.UK Page Builder*
10. A little software will appear where you will need a URL of a GOV.UK page and an [Anthropic API Key](https://platform.claude.com/).
11. Lastly, select the *Build Page* button and then **hold on to your butts...**
