# GOV.UK Figma Plugin - V1

## About
This is a test using Anthropic AI to analyze a screenshot of a GOV.UK page that will then be recreated in Figma using components from the GOV.UK Design System Figma UI Kit.

## How to run
1. Download this repo
2. Open Figma (ideally desktop version)
3. Select *Plugins* in the menu
4. Hover over *Development*
5. Choose the option *Import plugin from manifest*
6. Select the folder of downloaded repo
7. Select `manifest.json`
8. A pop-up should appear that contains all the plugins you've installed
9. Select *GOV.UK Page Builder*
10. A little software will appear where you will need a URL of a GOV.UK page and an [Anthropic API Key](https://platform.claude.com/).
11. Lastly, select the *Build Page* button and then hold on to your butts...
